4 Supporting Lemmas

Before proving the main universality result, we establish several key lemmas that characterize how TM operations affect the encoded sequences.

Lemma 5 Encoding Difference at Write
✓
#

Uses

  • Definition 1

Lean declarations

  • LeftTM0.encode_diff_at_write

The difference in encoding when writing at a position is \(\pm 2^k\) where \(k\) is the absolute position.

Lemma 6 Sequence Difference is Power of Two
#

Uses

  • Lemma 5

Lean declarations

  • LeftTM0.sequence_diff_is_power_of_two

One step of a TM changes the encoding by \(0\) or \(\pm 2^k\).

Lemma 7 K Value Equals Position
#

Uses

  • Lemma 6

Lean declarations

  • LeftTM0.sequence_k_equals_position

The \(k\) value in a sequence change equals the absolute position where the write occurred.

Lemma 8 K Value Bound
#

Uses

  • Lemma 7

Lean declarations

  • LeftTM0.sequence_k_bound

The \(k\) value is bounded by the step number: \(k \leq t\).

Lemma 9 Movement Constraint
#

Uses

  • Lemma 7

Lean declarations

  • LeftTM0.sequence_k_movement_constraint

Movement constraint between \(k\) values: \(|k_j - k_i| \leq j - i\).

Lemma 10 Natural Difference Zero Cases
✓
#

Uses

  • Lemma 5

Lean declarations

  • LeftTM0.encode_nat_diff_zero_cases

When the natural subtraction of encodings equals zero, either the same value was written or true was overwritten with false.

Lemma 11 Same Value Zero Difference
✓
#

Uses

  • Lemma 5

Lean declarations

  • LeftTM0.encode_same_value_zero_diff

Writing the same value as currently exists results in zero encoding difference.
